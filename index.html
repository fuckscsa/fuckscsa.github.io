⚠ A note on papers from the old system (pre-2016) ⚠

You must apply caution when using papers from the old system.

If the course has remained similar today, the papers can be reused. For example, chemistry has some good questions useful today.

It is not recommended to use papers in a course that has undergone large changes. At best, the individual questions may be used selectively

Additionally, some subjects may have split into multiple subjects so you must check the other related subjects for past papers. For example, mathematics methods and applications existed under the mathematics course as different stages.
According to SCSA, in this case, the previous courses have no similarity with current courses (it is incorrect to assume, say, one stage translates to applications and one to methods). So please be careful.

Stage 3 papers are the most useful for most subjects because stage 2 reflects a lower difficulty.
